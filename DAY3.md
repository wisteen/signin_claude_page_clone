Start the API: `python -m uvicorn api.main:app --reload --port 8000`
Start the frontend: `npm run dev`
Open the form at `http://localhost:5173/`
Open the admin panel at `http://localhost:5173/admin`
The simulator stores only email and timestamp.
Training codes are used for lab flow only and are not stored.
Passwords are not collected, stored, displayed, or logged.
Set the form to Stopped in `/admin` to block new submissions.
